Take a Bullet For the Country

By Pierce McBride

This project is an attempt to coallese and visulize gun violence

"Take a Bullet For the Country" scrapes data from the Gun Violence Archive "Last 72 Hours" page and filters that data based on the location of the user when they open the web page. The web page generates a bullet per gun-related incident and renders that data to the screen.

Outside Resources Used - A-Frame Jquery SocketIO "babyparse": "^0.4.6", "express": "^4.14.0", "jsonfile": "^2.4.0", "node-geocoder": "^3.15.1",

The project was created at Georgia Institute of Technology in the Digital Media graduate program. It was created as the third and final project in LMC 6310 The Computer as an Expressive Medium taught by Jay Bolter.
